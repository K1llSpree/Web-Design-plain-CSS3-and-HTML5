    • Programming With C#

    Introduction

  • C# (pronounced "C-sharp") is an object-oriented programming language from Microsoft that aims to combine the computing power of C++ with the programming ease of Visual Basic. C# is based on C++ and contains features similar to those of Java.

  • C# is designed to work with Microsoft's .NET platform. Microsoft's aim is to facilitate the exchange of information and services over the Web, and to enable developers to build highly portable applications. C# simplifies programming through its use of Extensible Markup Language (XML) and Simple Object Access Protocol (SOAP) which allow access to a programming object or method without requiring the programmer to write additional code for each step. Because programmers can build on existing code, rather than repeatedly duplicating it, C# is expected to make it faster and less expensive to get new products and services to market.

    • Hello World

    The first thing we want to know is how to write the first most common command in the world. The Hello World, in this case we want to use the command that is write for telling the machine that it has to present the data inside the quotation mark.

    using System;
      namespace ConsoleApp1 {
       class Program {
        static void Main(string[] args) {
         Console.WriteLine("Hello World!");
        }
       }
     }

  • In the code located above, we can see some elements that are new for us, the first thing is the Console WriteLine(""); that provides the console the ability of making the console to write the elements between the quotation mark.

    • Expressing Number in C#

    Numbers in C# can be expresses through variables using them as a container or pointers to make, to make this possible, the programmer has to make the initialization and also the definition of the variable with the integer or the double inside.

    using System;
      namespace ConsoleApp1 {
       class Program {
        static void Main(string[] args) {
         int a = 18;
         int b = 6;
         int d = a + b;
         Console.WriteLine(d);
        }
       }
     }

    As we see in the example, the numbers are located inside the variables with the name b and a, and there's also an operation with those variables that has a pointer in the d variable.
